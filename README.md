# Enciclopedia-Inanis

## Comandos do Git
- Desfazer todas as alterações: git reset --hard HEAD
- Atualizar local (**sempre rodar antes de falzer alterações**): git pull
- Adicionar TODAS as alterações: git add .
- Adicionar alterações de 1 arquivo: git add caminho/do/arquivo.md
- Adicionar alterações em 1 pacote: git commit -m "Texto descritivo sobre o pacote"
- Subir alterações: git push
